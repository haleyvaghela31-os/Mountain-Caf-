# La Montagna Digital Dream

A luxury Italian mountain café website built using Lovable AI, featuring authentic Italian cuisine, table reservations, customer feedback, and a modern responsive design.

## Features

* Interactive menu with prices and ingredients
* Table reservation system
* Customer reviews and feedback
* Gallery section
* Dark mode support
* Fully responsive design

## Tech Stack

* Lovable AI
* React
* TypeScript
* Tailwind CSS
* Framer Motion
